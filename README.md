# AVPlayer Fade

## About

The app demonstrates how to fade volume using AVPlayer.

## Requirements

- iOS 12+
- Swift 5.0

## Usage

1. Copy **AVPlayerExtensions.swift** to your project
2. Setup your AVPlayer and call the **fadeVolume** function

For more detailed explanation, check article:
